# Welcome to BOB - Building on Bitcoin

BOB is a Substrate-based and Bitcoin-optimized rollup.

BOB is a collection of software components enabling writing safe, Turing-complete, and Bitcoin-interoperable smart contracts in Rust and Solidity. By being fully EVM-compatible offering users seamless UX switching between EVM-native and Rust-native projects using existing wallets and tooling.

BOB is a common good. It's open-source nature serves to help onboard billions of people to Bitcoin by allowing builders a mighty tool to craft new applications.
